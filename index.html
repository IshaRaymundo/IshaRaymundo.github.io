    Sistema gestor de trabajos de mecánicos

    Descripción:

    Este sistema permite el acceso a los usuarios registrados gestionar todos sus trabajos, de tal forma que puedan ver cuales están en proceso y cuales ya se han terminado. El sistema cuenta con una lógica para calcular los precios totales dependiendo del tipo de trabajo, el precio de los materiales utilizados y las horas invertidas. Además de que se implementaron diversas medidas de seguridad y técnicas de autorización para una experiencia segura.

    Tecnologías utilizadas: JavaScript, TailwindCSS, React, y Express.js
